# Projet : Mini CRM

## Collaborateurs : FAZER Nino - TRAN Huu-Nghia

Mini-CRM est une application minimale de gestion de contacts développée en Go.

# Fonctionnalités

Gestion des contacts (Créer, Lire, Mettre à jour, Supprimer)
Structure de projet légère et extensible

# Prérequis

Go 1.18+ installé
Git

# Exécution

Lancer l'application en local :
go run .

Suivre l'application
